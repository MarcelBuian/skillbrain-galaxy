# In Js avem 2 tipuri de functii:

## Functia clasica:

```
function aduna_doua_numere(a, b) {
    return a + b;
}

const c = aduna_doua_numere(4, 2);
console.log("c = ", c);
```

## Functia sageata (arrow) =>

```
const aduna_doua_numere(a, b) => a + b;

const c = aduna_doua_numere = (4, 2);
console.log("c = ", c);
```
